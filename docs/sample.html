If you are trying to run this sample from the server, it should work right now - check out the sample.

If you are trying to run the sample yourself, there are some things for you to do:

  • This sample will not work from local file access on your local machine, it has to be placed on a webserver. (Of course your own webserver running on your local machine is fine)
  • Modify the /sample/sample.html file to place your OWN Google Maps API key into the request for the Google Maps API
  • Create your own database server with the Sample Data loaded into a database.
  • Modify /code/googleMapDBChunkerQuery.php to use your database with your username and password
  • Modify /code/googleMapDBChunkerCache.php to either disable the cache or nominate a directory for it to use
